# 會計事務所網站

本專案為一個簡單的會計事務所官方網站範例，主要用於展示事務所的公司介紹、服務內容以及聯絡資訊。網站整體設計以簡潔、清晰為主，方便使用者快速了解事務所提供的服務。

## 專案簡介

本網站為靜態網頁專案，主要使用 HTML 與 CSS 建立基本的企業形象網站。
透過簡單的版面設計與清楚的資訊架構，呈現會計事務所的專業形象。

## 專案功能

* 事務所簡介
* 服務項目介紹
* 聯絡資訊頁面
* 簡潔的網站版面設計

## 使用技術

* HTML5
* CSS3

## 使用方式

1. 下載或 clone 此專案
2. 使用瀏覽器開啟 `index.html` 即可查看網站內容

本專案為靜態網站，不需要額外安裝任何環境即可執行。

## 專案用途

本專案主要作為前端練習與網站設計練習使用，模擬會計事務所企業網站的基本架構與頁面設計。
